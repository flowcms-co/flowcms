## Highlights

This release refines the rich text editor's link tool and gives the whole editor toolbar one consistent icon set, and it fixes column alignment on the Content list so the SEO and Updated values line up under their headers.

## What's new

- A cleaner way to add links: clicking the link button in any rich text editor now opens a small inline panel where you paste or edit the URL, open it in a new tab, or remove the link. This replaces the old browser pop-up box, so adding and editing links stays inside the editor.
- A consistent toolbar: every formatting button in the editor toolbar and the inline selection bar now uses one matching icon set drawn at the same size and weight, so the toolbar reads as a single, uniform set instead of a mix of icons and characters. The link icon in particular is now clear and easy to see.

## Fixes and improvements

- Content list columns line up: the SEO score and the Updated date now sit directly under their column headers. Before, the header labels could drift out of line with the values in each row.
- General stability and performance improvements.
