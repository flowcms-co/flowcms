## v1.4.7: mobile header and drawer polish

Two small fixes for the mobile and installed (Add to Home Screen) Studio app.

- **Roomier header.** The mobile top bar now adds a little vertical breathing room (the status-bar
  safe-area inset plus a small offset, and bottom padding) so it is not cramped right under the
  status bar in the installed app. Desktop is unchanged.
- **Full sidebar in the mobile drawer.** Sidebar collapse is a desktop space-saver, but the saved
  collapse state was also shrinking the mobile slide-in drawer to an icon-only rail. Collapse now
  applies only on desktop, so opening the menu on a phone always shows the full, labelled sidebar.

Studio-only, CSS and shell changes; no API or behavior changes.
