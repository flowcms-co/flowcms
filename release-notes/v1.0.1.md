## Highlights

The first public release of FlowCMS: a headless CMS you can self-host, with a visual page builder, reusable components, and safe draft-then-publish editing.

## What's new

- Page builder with reusable components: compose pages from blocks and shared components.
- Draft over published: edit a live entry safely as a draft, then publish when you are ready, without disturbing what visitors currently see.
- One-click self-host upgrade: move to a new version from inside the app.

## Fixes and improvements

- Cleaner global structured data on fresh installs.
- More resilient installer: re-runs no longer fail on a root-owned .env, with a DNS pre-check and clearer guidance when a health check fails.
