# v1.7.1: a guided tour that points where it should

The guided tour is many people's first minutes with Flow CMS, and on larger
and scaled displays those minutes looked rough: spotlight rings drifting off
their targets, stray blue lines outside the highlight, and tour cards sitting
on top of the very thing they were describing. This release makes the tour
overlay precise on every screen size, on every page.

## The spotlight lands exactly on the target

- **The ring tracks its target live.** The overlay used to measure the target
  once and only re-measure on scroll or resize, so anything that shifted the
  layout afterwards (data loading in, animations settling) left the ring
  stranded where the element used to be. It now re-measures every painted
  frame, with interval and listener fallbacks, so the ring follows the target
  through layout shifts, window resizes and zoom changes.
- **Nothing paints outside the viewport.** Oversized targets (the full-height
  sidebar, full-width dashboard bands) are spotlit only where they are
  visible; the ring is clamped and whole-pixel aligned, which also removes the
  hairline seams that fractional display scaling produced.
- **The halo pulse stays put.** The breathing highlight was scale-based, so on
  wide targets its border floated tens of pixels outside the ring. It is now a
  fixed-spread ripple that looks the same on a small button and a full-width
  panel.

## The card never covers what it points at

Card placement now tries below, above, right, then left of the target, and
takes the first spot that fits fully on screen without touching the
spotlight. When none fits, the tour first nudges the page scroll to create a
clear strip; only for targets genuinely larger than the screen does the card
tuck into the far corner, over an edge, never over the middle. The card's
measured height also stays in sync while steps animate, which was the root
cause of cards hanging over the highlight.

Verified with an automated pass over all 13 tour chapters at three desktop
sizes: 96 of 96 placement and alignment checks pass.

## Also fixed

- The assets library no longer shows a stray lowercase "all" folder chip next
  to "All assets" in workspaces that have unfoldered media, and the duplicate
  React key warning that came with it is gone.

## Notes

Studio only. No migrations, no config changes, no new environment variables.
