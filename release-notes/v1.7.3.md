# v1.7.3: see the consent trail

v1.7.2 started recording evidence with every Terms acceptance — when it
happened, the request IP the server saw, the public IP the browser
reported for itself, and the parsed browser, OS and device. This release
makes that trail visible.

## Consent records, on the Security screen

Settings → Security now shows a Consent records table for admins: every
acceptance in the workspace with the person, what they accepted (terms,
product emails), the timestamp, both IPs and the device. Acceptances made
on v1.7.2 already carry the full evidence and appear here immediately
after updating.

## Notes

Studio and API. No migrations, no config changes, no new environment
variables.
