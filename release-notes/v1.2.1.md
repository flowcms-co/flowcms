## Highlights

This release makes editing richer and the live editor smarter. You get a full rich-text toolbar with inline images, you can add and remove repeatable items like FAQs and services right on your live page, and you can schedule publishing from everywhere you work.

## What's new

- Richer text editor: a full toolbar with inline images (from your library, a URL, or drag-and-drop and paste), emoji, text color and highlight, alignment, task lists, tables, code blocks, and more. The same tools work in the main canvas and in nested fields.
- Add and edit list items live: on your live page you can now add, edit, and remove repeatable items such as FAQs, services, and testimonials, not just change text that is already there.
- Schedule publishing anywhere: set a publish time from the editor, the content row menu, or the bulk action bar.
- Smarter previews for new pages: when you preview a page you have not published yet, FlowCMS borrows the layout of an existing published page of the same type, so you see a real frontend to edit instead of an error. A banner tells you when a template is being used.
- Cleaner field mapping: slugs, page titles, SEO and meta tags, and internal ids no longer clutter the field mapper or the live editor, so you only see the content that actually appears on the page.
- Tidier dashboard: the overview is back to a four-card view with trend sparklines and change indicators, and Recent activity now shows real author avatars.
- Bulk duplicate: copy several entries at once from the content list.
- Wider layout: the app now uses more of the screen on large displays.

## Fixes and improvements

- Slugs are now unique per content type and language, fill in automatically from the page title, and auto-number when there is a conflict, while staying fully editable.
- Field and content-type keys are normalized to consistent names automatically, which keeps your API names tidy.
- General stability and performance improvements.
