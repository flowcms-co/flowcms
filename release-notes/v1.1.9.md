## Highlights

This release makes in-app upgrades more reliable and keeps your server's disk from filling up over time.

## Fixes and improvements

- Automatic cleanup after updates: once an update succeeds, old version files are removed automatically, so past versions no longer pile up and fill your server's disk.
- Clearer messages: when an update cannot complete, FlowCMS now shows the real reason (for example, running low on disk space) instead of a confusing "image not found" message.
- Resilient downloads: updates retry automatically if a download is briefly interrupted, so a momentary network hiccup no longer stops an upgrade.
- Tidier backups: the backups list no longer shows a stray "Invalid Date" entry.
- More compatible update files: images are published in a simpler, widely supported format.
- General stability improvements.

As always, every upgrade takes a full backup first and rolls back automatically if anything goes wrong, so your site stays safe.
